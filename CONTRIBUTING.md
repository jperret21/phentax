## Contributing to PhenTAX

Contributions to `phentax` are encouraged and appreciated! We use the [GitHub issues tracker](https://github.com/asantini29/phentax/issues) to manage and discuss contributions.

To contribute, please follow these steps:
1. Fork the repository and create a new branch for your feature or bug fix.
2. Make your changes and commit them with clear messages.
3. Push your changes to your fork and open a pull request against the main repository. Please, include minimal but appropriate tests.
